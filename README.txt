当当网图书畅销榜数据分析系统
=========================

1. 程序启动
-----------------
执行以下文件启动程序：
src/main.py

启动命令：
python src/main.py

2. 运行说明
-----------------
本程序提供图形界面，启动后可以看到三个功能标签页：

a) 数据采集
- 选择"数据采集"标签页
- 设置采集页数（例如：1-10）
- 点击"开始采集"按钮
- 等待采集完成

b) 数据分析
- 切换到"数据分析"标签页
- 选择分析类型
- 点击"开始分析"按钮
- 查看分析结果

c) 数据导出
- 切换到"数据导出"标签页
- 选择导出格式
- 点击"导出"按钮
- 选择保存位置

3. 环境要求
-----------------
* Python版本：3.8或更高版本

* 必需的第三方库：
  - requests>=2.31.0    (网络请求)
  - beautifulsoup4>=4.12.0 (网页解析)
  - pandas>=2.0.0       (数据处理)
  - numpy>=1.24.0       (数据计算)
  - matplotlib>=3.7.0   (数据可视化)
  - seaborn>=0.12.0     (统计可视化)
  - PyQt5>=5.15.0       (图形界面)
  - python-dateutil>=2.8.0 (日期处理)
  - openpyxl>=3.1.0     (Excel支持)

安装依赖：
pip install -r requirements.txt

4. 运行示例
-----------------
# 启动程序
python src/main.py

# 数据采集示例
1. 在"数据采集"标签页
2. 输入采集页数：5
3. 点击"开始采集"
4. 等待完成，查看日志信息

# 数据分析示例
1. 切换到"数据分析"标签页
2. 选择"销量排行榜"
3. 点击"开始分析"
4. 查看生成的图表

# 数据导出示例
1. 切换到"数据导出"标签页
2. 选择"Excel格式"
3. 点击"导出数据"
4. 选择保存位置
5. 导出文件名：book_data_20240101.xlsx

注意：首次运行时，程序会自动创建必要的目录结构（data/和logs/）。
